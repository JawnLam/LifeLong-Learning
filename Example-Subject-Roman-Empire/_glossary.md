# Roman Empire — Glossary

Canonical terms. A term is added **only after** the learner has demonstrated understanding of it — this file is a record of earned, compressed knowledge, not a dictionary written up front. One canonical term per concept; discouraged alternatives noted.

## Terms

### Cursus honorum
The fixed sequence of public offices (quaestor → aedile/tribune → praetor → consul), with minimum ages, through which a senatorial career advanced. *Preferred over* "career ladder," which loses the legal-constraint sense.

### Mos maiorum
The unwritten "way of the ancestors" — the body of traditional norms that constrained Roman political behavior more powerfully than written law.

### Imperium
The formal power of military and civil command held by senior magistrates — distinct from *auctoritas* (informal prestige/influence). *Discouraged:* using "power" undifferentiated where the *imperium* vs *auctoritas* distinction is load-bearing.
